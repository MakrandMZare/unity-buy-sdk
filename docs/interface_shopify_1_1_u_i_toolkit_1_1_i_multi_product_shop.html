Called when products have been loaded by the controller. This usually happens asynchronously after you have told the controller to load products via ShopController.LoadProducts()

Parameters
productsThe products that were loaded, the length of the array will match the count sent to ShopController.LoadProducts()
afterBecause products load in pages, after is returned from the GraphQL API with the product ID at the end of the fetched page.
